# Responsive Task Manager

A modern responsive task manager built using **HTML, CSS, and JavaScript**.  
This application allows users to manage daily tasks efficiently with a clean UI and persistent storage.

## Features

- Add new tasks
- Mark tasks as completed
- Delete individual tasks
- Clear all tasks
- Task counter for remaining tasks
- Local Storage support (tasks persist after refresh)
- Responsive design for mobile and desktop
- Smooth UI animations
- Add tasks quickly using the **Enter key**

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API

## How It Works

- Tasks are stored using the browser's **Local Storage**.
- When the page loads, saved tasks are retrieved and displayed.
- Users can mark tasks complete, delete them, or clear all tasks.
- The UI updates dynamically using JavaScript.
